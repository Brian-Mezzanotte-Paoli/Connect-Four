# Run
  `python main.py [mode] [starter] [emule]`
  - **mode**
    - [**pp**] player versus player
    - [**pc**] player versus computer
    - [**cc**] computer versus computer

  - **starter**:
      - [**1**] start player 1
      - [**2**] start player 2

  - **emule**:
      - [**0**] use with machince
      - [**1**] use without machine
