# Run
  `python main.py [mode] [starter] [emule]`
  **mode** -> pp -> player versus player
              pc -> player versus computer
              cc -> computer versus computer
  **starter** -> 0 / 1: who start the game
  **emule**-> 0 / 1: boolean if use the machine
